[활용 가능한 데이터를 제공하는 Open API 사이트]

서울시 열린 데이터 광장
 - http://data.seoul.go.kr/

우체국
버스운송사업조합
기상청

[1] 회원가입
[2] API 신청(필요하다면 신청서를 작성)
[3] 승인을 받으면 인증키가 발급
[4] 요청 주소로 신호를 보낼 때 인증키값을 첨부
[5] 결과를 수신하여 parsing 수행
 - xml	html과 유사한 태그 형식으로 정보를 표현
 - json	자바스크립트 양식으로 정보를 표현


<xml과 json 형식의 비교>

[데이터 샘플]
사람1 - 이름:홍길동, 나이:20, 성별:남자
사람2 - 이름:장희빈, 나이:30, 성별:여자

[데이터 샘플을 xml방식으로 표현] - Web 방식의 탐색 용이
<사람목록>
    <사람>
        <이름>홍길동</이름>
        <나이>20</나이>
        <성별>남자</성별>

    <사람>

        <성별>여자</성별>
    </사람>
</사람목록>
